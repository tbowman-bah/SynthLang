# Open Sets
(X,τ) - topological space
U ∈ τ ⟹ U is open

# Continuity
f: X → Y is continuous
∀V∈τY: f⁻¹(V)∈τX

# Connectedness
X = A ∪ B, A∩B = ∅
A,B open ⟹ X disconnected
